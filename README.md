# 📝 Quick Notes - Chrome 快捷笔记插件

> 随时记录灵感与待办事项，功能丰富、轻量快速、开源。

## 功能特点

- ✅ **笔记管理**：添加 / 删除笔记，存入 Chrome 本地
- 🔍 **快速搜索**：输入即搜，实时过滤笔记内容
- 🌙 **深色模式**：一键切换，明暗主题自动记住
- 📌 **置顶笔记**：重要笔记置顶显示，方便查找
- 📤 **导出笔记**：一键导出为 JSON 文件备份
- 📥 **导入笔记**：从 JSON 文件恢复笔记，支持增量导入
- 📝 **字符统计**：每条笔记和全部笔记的字符数一目了然
- ⌨️ **快捷键**：Enter 保存笔记，Ctrl+Enter 换行
- 💾 **本地存储**：所有数据保存在 Chrome 本地，不上传服务器

## 安装方式

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 右上角开启 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本项目文件夹即可

## 项目结构

```
chrome-extension/
├── manifest.json      # 插件配置（Manifest V3）
├── popup.html         # 弹窗主页面（含深色模式样式）
├── popup.js           # 交互逻辑（搜索/置顶/导入导出等）
├── background.js      # 后台 Service Worker
└── icons/             # 图标资源（16/48/128）
```

## 快捷键

| 按键 | 功能 |
|------|------|
| `Enter` | 保存笔记 |
| `Ctrl + Enter` | 换行 |
| `Ctrl + F` | 聚焦搜索框 |

## 数据说明

- 笔记存储在 `chrome.storage.local` 中
- 主题偏好（深色/白天）独立存储
- 导出文件为标准 JSON，可在任意文本编辑器中查看
- 导入支持增量合并（自动跳过重复笔记）
